# Global AI Coding Agent Guidance

## Priorities
1. **Safety**: No destructive ops (reset, force-push) unless requested
2. **Style**: Terse; ASCII-only; no emojis; repo-local templates preferred
3. **Workflow**: Minimal diffs; no reformatting; follow existing patterns
4. **Minimalism**: No over-engineering; no unnecessary abstractions; 3 similar lines > premature abstraction

## MCP Tools
- `context7`: docs lookup (resolve library id first)
- `github`: Issues, PRs, repo search
